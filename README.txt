Up-to-date instructions can always be found at: https://nitrox.fandom.com/wiki/Nitrox_Wiki

1) Extract ALL files from this zip file into a clean directory of your choosing. 
2) Double click NitroxLauncher.exe to start the launcher.
   - If you encounter Windows SmartScreen errors click 'More Info' -> 'Run Anyway'
3) Ensure the path to the game is correct in the 'Options' tab of the launcher
4) Start the Nitrox server in the 'Server' tab of the launcher.
   - Only one player needs to host the server.  More information can be found here: https://nitrox.fandom.com/wiki/Connecting_to_Server
5) Start the game with the 'Play Multiplayer' button on the 'Start Game' screen.
6) Connect to a server.  Instructions can be found here: https://nitrox.fandom.com/wiki/Connecting_to_Server
